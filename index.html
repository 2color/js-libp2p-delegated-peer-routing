Intro

Installable via npm install --save libp2p-delegated-peer-routing, it can also be used directly in the browser.

Download

The source is available for download from GitHub. Alternatively, you can install using npm:

$ npm install --save libp2p-delegated-peer-routing

You can then require() libp2p-delegated-peer-routing as normal:

const libp2PDelegatedPeerRouting = require('libp2p-delegated-peer-routing')

In the Browser

libp2p-delegated-peer-routing should work in any ECMAScript 2018 environment out of the box.

Usage:

<script type="text/javascript" src="index.js"></script>

The portable versions of libp2p-delegated-peer-routing, including index.js and index.min.js, are included in the /dist folder. libp2p-delegated-peer-routing can also be found on unpkg.com under

  • https://unpkg.com/libp2p-delegated-peer-routing/dist/index.min.js
  • https://unpkg.com/libp2p-delegated-peer-routing/dist/index.js
